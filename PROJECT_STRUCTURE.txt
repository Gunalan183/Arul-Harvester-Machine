DETAILED FILE BREAKDOWN
========================

Backend Files (Server)
----------------------

server/server.js (Entry Point)
├── Express app setup
├── Middleware configuration (CORS, JSON)
├── MongoDB connection
├── Route mounting
└── Server listening

server/controllers/authController.js
├── login() - User authentication
└── register() - User registration

server/controllers/customerController.js
├── getAllCustomers() - List all customers
├── getCustomerById() - Get single customer
├── getCustomerBills() - Get customer's bills
├── createCustomer() - Add new customer
├── updateCustomer() - Edit customer
└── deleteCustomer() - Remove customer

server/controllers/billController.js
├── getAllBills() - List bills with filters
├── getBillById() - Get single bill
├── createBill() - Create new bill
│   ├── Calculate total minutes
│   ├── Calculate total amount
│   └── Auto-increment bill number
├── deleteBill() - Remove bill
└── getNextBillNumber() - Get next bill number

server/controllers/reportController.js
├── getDashboardStats() - Dashboard statistics
├── getDailyReport() - Daily report
└── getMonthlyReport() - Monthly report

server/models/User.js
├── username (String, unique)
└── password (String, hashed)

server/models/Customer.js
├── name (String, required)
├── phone (String)
└── village (String)

server/models/Bill.js
├── billNumber (Number, unique)
├── date (Date)
├── customer (ObjectId ref)
├── acres (Number)
├── startTime (String)
├── endTime (String)
├── totalMinutes (Number)
├── ratePerHour (Number)
└── totalAmount (Number)

server/middleware/auth.js
├── Extract JWT token
├── Verify token
└── Attach user to request

Quick Start Commands
--------------------

# Install dependencies
npm install && cd client && npm install && cd ..

# Start development
npm run dev

# Create admin user
curl -X POST http://localhost:5000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}'

# Open browser
http://localhost:3000
